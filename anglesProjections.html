Angles and Projections

Para poder estimar una posición, se pueden utilizar diversas técnicas, pero todas comparten el mismo principio matemático. También, depende del caso y los datos que uno tenga. En este ejemplo, se tienen dos puntos y se obtendrá su proyección. Para lograr este objetivo es necesario saber la dirección de movimiento y esto se obtiene con el ángulo de inclinación de un punto respecto a otro. Se emplea tanto el coseno y el seno del ánglo más una distancia a proyectar que dependerá de las circunstancias (se pueden emplear diferentes variables, en este ejemplo sencillo se da un valor fijo). Como se observa no es muy complicada esta labor y es de gran ayuda.
